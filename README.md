# InfiniteSight
## 文件树图
```
InfiniteSight/
├── .venv/                  # 虚拟环境
├─ themes/                  # 样式文件
│  ├─ dark.qss              # 深色样式
│  └─ light.qss             # 浅色样式
├── i18n/                   # 国际化文件
│   ├── en_us.json          # 英文翻译
│   └── zh_cn.json          # 中文翻译
├── main.py                 # 入口文件
├── image_cache.py          # 图像缓存工具
├── image_loader.py         # 图像加载器
├── image_viewer.py         # 图像查看器主窗口
├── language_manager.py     # 语言管理器
├── settings.py             # 设置管理器和对话框
└── vips/                   # libvips 预编译二进制文件
    └── bin/                # libvips 的 DLL 文件
```
